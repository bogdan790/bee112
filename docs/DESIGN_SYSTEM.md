# Design System – BEE112

## COLOR USAGE

Primary color:
- Existing BEE112 green (do not change)

Backgrounds:
- White
- Very light grey / very light green tint

Avoid:
- Strong contrasts
- Saturated colors
- Aggressive gradients

## TYPOGRAPHY

- Existing font family only
- Minimum font size on mobile: 16px
- Clear hierarchy between headings and body text

## SECTIONS

- Consistent vertical padding
- Clear separation between sections
- Alternate subtle background tones to aid readability

## GOAL

The site should feel:
- Calm
- Clear
- Trustworthy
- Easy to read, especially on mobile

# BEE112 – Agent Brief (READ FIRST)

Project: bee112.ro  
Purpose: calm, phone-first apicultural support

This project is NOT a generic website.
It is a public-facing support service focused on clarity, calm, and trust.

## NON-NEGOTIABLE PRINCIPLES

- Phone-first UX (calling is the main action)
- Calm tone (no panic, no sensationalism)
- Clarity over visual effects
- Predictable UX patterns
- One primary CTA per screen

## FORBIDDEN (ABSOLUTELY NO)

- Empty buttons
- Icon-only CTAs
- Duplicate CTAs on the same screen
- Repeating “Ai găsit un roi de albine?” on every page
- Aggressive colors or gradients
- Visual clutter on mobile
- UX experiments without approval

## BEFORE MAKING ANY CHANGE

You MUST read and respect:
- docs/CTA_RULES.md
- docs/UX_RULES.md
- docs/DESIGN_SYSTEM.md

If a requested change conflicts with these rules:
STOP and report the conflict.

These rules apply to ALL pages.

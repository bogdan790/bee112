# BEE112.ro

## Ce este
BEE112 este o initiativa civica de voluntariat pentru salvarea roiurilor de albine din mediul urban
(Bucuresti si Ilfov).

## De ce exista
Pentru a proteja albinele, a educa publicul si a oferi o alternativa civilizata la distrugerea lor.

## Ce NU este
- Nu este serviciu comercial
- Nu este call-center
- Nu este interventie de urgenta tip 112

## Public tinta
- Cetateni speriati / neinformati
- Administratii / institutii
- Presa si ONG-uri
